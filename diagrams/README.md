# Diagrams

## Open locally

- Open `diagrams/architecture.html` in any browser.

## Export to PDF (for Medium / CFP submissions)

- In the browser: **Print… → Save as PDF**
- Recommended: set paper to **Landscape** and disable headers/footers if possible.

## Notes

- The diagram is **self-contained**: pure HTML + CSS + inline SVG (no CDN, no images).
